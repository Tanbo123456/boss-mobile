# 源码项目说明:
	boss_client: 前台项目
	boss_server: 后台项目
	

# react项目-boss_client的运行说明
## 1. 准备
	1) 确保安装了node环境
		查看是否已经安装: node -v

	2) 确保安装了mongodb, 并启动了对应的服务
		查看是否安装并启动了服务: 右键-->任务管理器-->服务-->MongoDB
		
	
## 2. 启动后台应用
	1). 进入boss_client
	2). 执行命令: npm start

## 3. 启动前台应用并访问
	1). 进入boss_server
	2). 执行命令: npm start
